# Ecoarchi Website

Website landing page modern untuk **Ecoarchi**, UMKM lokal yang berfokus pada produk ramah lingkungan berbasis material daur ulang.

## Fitur

* Responsive design (desktop, tablet, mobile)
* Navigasi mobile hamburger menu
* Hero section modern
* Product showcase
* Impact statistics
* Contact form simulation
* Arsitektur kode modular ala Ponytail

## Teknologi

* HTML5
* CSS3
* Vanilla JavaScript
* BEM Naming Convention
* Design Tokens Architecture

## Struktur Project

```bash
ecoarchi/
│
├── index.html
│
├── css/
│   └── style.css
│
├── js/
│   └── main.js
│
└── README.md
```

## Menjalankan Project

Clone repository:

```bash
git clone https://github.com/USERNAME/NAMA_REPO.git
```

Masuk folder:

```bash
cd NAMA_REPO
```

Buka `index.html` menggunakan browser atau Live Server.
