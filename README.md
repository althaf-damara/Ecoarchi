# Ecoarchi Website
Platform produk daur ulang yang mengubah limbah menjadi karya bernilai guna, mendukung gaya hidup berkelanjutan dan ekonomi sirkular di Indonesia.

## Tentang Proyek
Ecoarchi adalah website one-page yang menampilkan berbagai produk ramah lingkungan dari bahan daur ulang. Website ini dibangun sebagai bentuk dukungan terhadap SDG 12 (Konsumsi dan Produksi yang Bertanggung Jawab) dengan mengedukasi pengunjung tentang pentingnya pengolahan limbah dan memberdayakan perajin lokal.

## Fitur Utama
1. Hero Section – Perkenalan singkat dan ajakan aksi untuk melihat produk.
2. Cerita Ecoarchi – Latar belakang dan misi keberlanjutan.
3. Grid Produk – 8 produk daur ulang dengan detail bahan dan tombol pemesanan.
4. Formulir Kontak – Kirim pesan langsung ke tim Ecoarchi (simulasi).
5. Animasi Scroll – Efek fade-in saat elemen muncul di layar.
6. Mobile Responsif – Navigasi hamburger untuk perangkat kecil.
7. Kredit Gambar – Footer mencantumkan sumber gambar secara transparan.

## Menjalankan Project
1. Clone repositori atau unduh semua file.
2. Pastikan folder assets berisi semua gambar yang diperlukan (lihat daftar di index.html).
3. Buka index.html di browser favorit Anda.

## Teknologi
1. HTML5 – Struktur semantik.
2. CSS3 – Grid, flexbox, variabel kustom, animasi.
3. JavaScript Vanilla – Tanpa library/framework.
4. Google Fonts – Inter untuk tipografi.

## Desain & Styling
1. Palet warna utama: hijau tua #2F5D50, latar #F7F6F2, dan abu-abu lembut #ECE9DF.
2. Breakpoint: mobile-first dengan titik henti di 1024px, 768px, dan 480px.
3. Konsistensi: semua tombol dan tautan memiliki efek hover yang halus.
